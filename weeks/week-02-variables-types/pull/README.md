- Họ và tên: Phan Văn Lực  
- Chủ đề: Biến, kiểu dữ liệu và input trong Python

## 📚 Giới thiệu

Đây là bài tập Python cơ bản nhằm giúp làm quen với cách khai báo biến, sử dụng kiểu dữ liệu, chuyển đổi dữ liệu, nhập dữ liệu từ bàn phím và thực hiện các phép tính đơn giản.

## 📦 Nội dung bài tập

### 1. Biến trong Python
- Khai báo biến với các kiểu dữ liệu:
  - `str`
  - `int`
  - `float`
  - `bool`
- Kiểm tra kiểu dữ liệu bằng `type()`
- Hoán đổi giá trị biến không dùng biến tạm
- Sử dụng toán tử gán (`+=`, `-=`, `*=`, `//=`)
- Gán nhiều biến trên một dòng

### 2. Chuyển đổi kiểu dữ liệu
- Chuyển đổi giữa `int()`, `float()`, `str()`, `bool()`
- Kiểm tra giá trị Boolean
- Tính chỉ số BMI
- Chuyển đổi số giây sang giờ, phút, giây

### 3. Máy tính nhận input
- Tính tổng, hiệu, tích, thương của 2 số
- Tính diện tích và chu vi hình tròn
- Tính giá sau giảm giá
- Đổi tiền từ VNĐ sang USD
